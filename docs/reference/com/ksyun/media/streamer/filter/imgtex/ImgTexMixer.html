Class Overview

Video frame mixer with openGLES.

Has maximum getSinkPinNum() input pins, the lower sink pin index number means the lower frame z order, that is, the higher sink pin index frame shown in top of lower ones.

The mixer operation occurred while the frame with main index number arrived.
